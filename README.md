# 🔐 Cybersecurity Basics

This repository serves as the **foundation** of my cybersecurity learning journey. Here I have documented the core concepts of cybersecurity — what it is, why it matters, and how it protects our digital world.

---

## 📌 What is Cybersecurity?

> *"Cybersecurity is the practice of protecting systems, networks, programs, and data from digital attacks, damage, or unauthorized access."*

In simple words — **Cybersecurity is the art of keeping the bad guys out while letting the good guys in.**

---

## 🔥 Why is Cybersecurity Important?

| **Reason** | **Explanation** |
| :--- | :--- |
| **Data Protection** | Protects sensitive information like personal details, financial data, and medical records |
| **Financial Loss Prevention** | Cyber attacks cost companies billions of dollars annually. A single attack can bankrupt a small business |
| **National Security** | Protects critical infrastructure like power grids, water systems, and defense networks |
| **Privacy** | Ensures that our private conversations, photos, and information stay private |
| **Business Continuity** | Prevents ransomware and DoS attacks from shutting down operations |
| **Reputation** | A single data breach can destroy a company's reputation built over years |

## 🎯 Main Goals of Cybersecurity

The CIA Triad is the foundation of cybersecurity:

| **Principle** | **Meaning** | **Example** |
| :--- | :--- | :--- |
| 🔒 **Confidentiality** | Only authorized people can access the data | Encryption, Passwords, Access Control |
| 📝 **Integrity** | Data cannot be tampered or altered without authorization | Hash functions, Digital Signatures |
| 🌐 **Availability** | Data and systems are available when needed | Backups, Redundancy, DDoS Protection |

## 🛡️ Common Types of Cyber Threats

| **Threat** | **What it does** |
| :--- | :--- |
| **Malware** | Malicious software (Virus, Worm, Trojan, Ransomware) |
| **Phishing** | Fake emails/SMS to steal credentials |
| **DDoS Attack** | Floods servers to make services unavailable |
| **Man-in-the-Middle (MITM)** | Intercepts communication between two parties |
| **SQL Injection** | Injects malicious code into databases |
| **Cross-Site Scripting (XSS)** | Injects malicious scripts into websites |
| **Zero-Day Attack** | Exploits unknown vulnerabilities |
| **Social Engineering** | Manipulates people to reveal information |

---

## 📚 Domains of Cybersecurity

| **Domain** | **What it covers** |
| :--- | :--- |
| **Network Security** | Securing network infrastructure (Firewalls, IDS/IPS, VPNs) |
| **Application Security** | Securing software and web applications (OWASP Top 10, SAST, DAST) |
| **Endpoint Security** | Securing end-user devices (Antivirus, EDR, Hardening) |
| **Cloud Security** | Securing cloud environments (AWS, Azure, GCP) |
| **Identity & Access Management** | Managing user identities and access control (MFA, SSO, RBAC) |
| **Incident Response** | Responding to and recovering from security breaches |
| **Disaster Recovery** | Business continuity after an attack |
| **Compliance & Governance** | Following regulations (GDPR, HIPAA, PCI-DSS, DPDP Act) |
| **Penetration Testing** | Ethically hacking to find vulnerabilities |
| **SOC Operations** | Security monitoring and threat hunting |
